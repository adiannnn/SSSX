# Windows防火墙配置指南 - 允许外部访问8000端口

## 方法一：通过Windows Defender防火墙设置

1. 打开控制面板 -> 系统和安全 -> Windows Defender防火墙
2. 点击左侧的"高级设置"
3. 在左侧面板中选择"入站规则"
4. 在右侧面板中点击"新建规则..."
5. 规则类型选择"端口"，点击"下一步"
6. 选择"TCP"，并在"特定本地端口"中输入"8000"，点击"下一步"
7. 选择"允许连接"，点击"下一步"
8. 保留默认的网络类型选择，点击"下一步"
9. 为规则命名（如"HTTP Server Port 8000"），并添加描述，点击"完成"

## 方法二：使用命令提示符（管理员权限）

1. 以管理员身份打开命令提示符
2. 执行以下命令：
   ```
   netsh advfirewall firewall add rule name="HTTP Server 8000" dir=in action=allow protocol=TCP localport=8000 remoteip=any profile=private,public,domain
   ```

## 注意事项

- 如果您在公共网络上，请谨慎开放端口
- 确保只在需要远程访问时运行服务器
- 完成使用后，建议删除或禁用此防火墙规则
- 如果您使用第三方防火墙软件，请参考其文档添加相应规则
